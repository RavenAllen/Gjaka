Mga pinagkunan (search results)

Ang modelo (TinyLlama 1.1B) ay tumatakbo gamit ang WebAssembly (CPU) via Transformers.js — walang ipinapadalang tanong o sagot papunta sa kahit anong server. Ang tanging kumokontak sa labas ay ang search step (Render), na naghahanap lang ng web results; ang pag-generate ng sagot ay nasa device mo. Kung masyadong mabagal sa phone mo, pwede mong palitan ang MODEL_ID sa loob ng file sa mas maliit na model (hal. onnx-community/Qwen2.5-0.5B-Instruct).
